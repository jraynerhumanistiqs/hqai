## What this is
HQ.ai is an AI-powered HR and recruitment SaaS product built under 
the parent brand Humanistiqs. It serves Australian SMEs (10-250 staff) 
with AI-driven HR compliance, document generation, recruitment support, 
and human advisor escalation.

## Tech stack
- Next.js 16 (App Router)
- Tailwind CSS
- Supabase (auth + database)
- Anthropic Claude API (claude-sonnet-4)
- Vercel (hosting)

## What's built and working
- Auth: login, signup, magic link
- 3-step business onboarding
- HQ People chat (Claude API, streaming)
- HQ Recruit chat (Claude API, streaming)
- Document auto-save to library
- Document library page
- Settings page
- Sidebar navigation
- Escalation detection with advisor card
